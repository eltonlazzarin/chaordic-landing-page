## :information_source: How To Use

```bash
# Clone this repository
$ git clone https://github.com/eltonlazzarin/chaordic-landing-page.git

# Go into the repository
$ cd chaordic-landing-page

# Install dependencies
$ yarn

# Run the app
$ yarn start
```
